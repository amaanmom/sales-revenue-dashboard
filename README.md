# sales-revenue-dashboard
Sales & Revenue Analysis Dashboard
A professional and interactive business intelligence dashboard built using HTML, CSS, JavaScript, and Chart.js. This project visualizes sales and revenue performance through dynamic charts, KPI cards, product analysis, and responsive design.

🚀 Features


📊 Interactive Revenue Trend Analysis


💰 KPI Cards for Business Metrics


🥧 Revenue by Category Doughnut Chart


📈 Top Products Revenue Visualization


📋 Product Performance Table


🎛️ Year & Region Filters


📱 Fully Responsive UI


⚡ Real-time Dynamic Updates


🎨 Clean Corporate Dashboard Design
